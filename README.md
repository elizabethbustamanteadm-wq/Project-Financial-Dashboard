# 🗽 NY Project Operations & Financial Dashboard

## 📊 Project Overview
This project demonstrates my ability to manage operational and financial data for large-scale projects in New York. I transformed raw data into an **Executive Dashboard** that enables real-time budget variance tracking and identifies potential financial risks.

## 🛠️ Tech Stack
* **SQL:** Data extraction, cleaning, and variance analysis.
* **Power BI:** Professional dashboard design and KPI visualization.
* **Excel/CSV:** Management of vendor datasets and project records.

## 🔑 Key Business Insights
* **Budget Variance Analysis:** Identified that the "Brooklyn Gov Hub" project exceeded its original budget by 15%.
* **Risk Management:** Flagged projects as "At Risk" or "Delayed" based on spending patterns and timelines.
* **Strategic Reporting:** Simplified complex date structures into an executive view for faster decision-making.
